Canvas Effects List

Below are the list of Canvas Effects. Please Click to view the effects.
  • Constellation Effect (Random) This effects shows moving dots, whose size increases when they are nearer to the mouse pointer. And they form a line when any two of the dots are nearby, whose width denotes the strength of the force.
  • HAppy Diwali 2016 (AutoTyping & Fireworks) This is a automatic firework for diwali using fireworks effects which is modified to add some extra effects. The wish message is using the AutoTyping module.
  • FireWorks This is a simple Fireworks effects. A firework is fired from the bottom center of the screen, and is blasted at the point of cursor click. If the mouse button is clicked and moved, different location in the path is stored, and simultaneously multiple fireworks will be fired to each location.
